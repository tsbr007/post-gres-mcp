PostgreSQL Manager + MCP Server — Setup & Deployment Instructions
================================================================

If you download this project onto a new machine/system, follow these steps to 
enable database connectivity, install dependencies, run the GUI, and configure
the MCP server for Claude Desktop.

--------------------------------------------------------------------------------
STEP 1: Prerequisites
--------------------------------------------------------------------------------
Ensure the following are installed on the new system:
1. Python (version 3.10 or higher)
   - Download from: https://www.python.org/downloads/
   - IMPORTANT: Check the box "Add Python to PATH" during installation.
2. PostgreSQL Database Server (either local or accessible over network).

--------------------------------------------------------------------------------
STEP 2: Installing Dependencies
--------------------------------------------------------------------------------
Open a terminal (cmd or PowerShell) in the project directory and run:

- On Windows:
    install.bat

- Or manually (any platform):
    pip install -r requirements.txt

This installs psycopg2-binary (driver), Faker (mock generator), sqlparse, and mcp.

--------------------------------------------------------------------------------
STEP 3: Configure Database Connectivity
--------------------------------------------------------------------------------
Open the file 'config.ini' in a text editor and configure your connection profile.
By default, the template is under [profile_local]:

[profile_local]
name = Local PostgreSQL
host = localhost             <-- Change to your database host IP/domain
port = 5432                  <-- Database port
database = postgres          <-- Target database name
username = postgres          <-- Database username
password =                   <-- Database password
ssl_mode = prefer            <-- SSL mode (prefer/require/disable)

You can copy and paste this section to create additional profiles (e.g., [profile_staging]).

--------------------------------------------------------------------------------
STEP 4: Run the Desktop GUI Application
--------------------------------------------------------------------------------
To open the user interface to browse the database, run queries, and mock data:

- On Windows:
    Double-click or run: run.bat

- Or manually:
    python main.py

--------------------------------------------------------------------------------
STEP 5: Configure Claude Desktop (MCP integration)
--------------------------------------------------------------------------------
To let Claude connect to your PostgreSQL database and run tools:

1. Open your Claude Desktop configuration file.
   - On Windows: Press Win+R, type '%APPDATA%\Claude\' and open 'claude_desktop_config.json'.
2. Add the following entry. Be sure to update the absolute paths to match the exact 
   location of the project on your new system:

{
  "mcpServers": {
    "postgres-mcp": {
      "command": "python",
      "args": ["C:/YOUR_NEW_PATH/postgres-mcp/mcp_server.py"],
      "cwd": "C:/YOUR_NEW_PATH/postgres-mcp"
    }
  }
}

3. Restart Claude Desktop. You will see a new tool plug-in icon in the input area.
